APPENDIX C

주요 메소드·함수 빠른 참조 카드

본문에서 한 번씩은 다룬 빈출 함수와 메소드를 자료형·모듈별로 표로 정리하였다. 시그니처는 입문 학습에 필요한 핵심만 단순화하였으며, 실전 코드를 작성할 때 곁눈질로 빠르게 확인하기 위한 카드 역할을 한다.

C.1 내장 함수

입출력·자료형 변환

시그니처설명
print(*objects, sep=' ', end='\n')표준 출력. 여러 인자를 sep으로 잇는다.
input(prompt='')한 줄 입력을 문자열로 반환.
int(x), float(x), str(x), bool(x)기본 형 변환.
list(iter), tuple(iter), set(iter), dict(pairs)컬렉션 생성.
type(x), isinstance(x, T)객체의 자료형 확인 / 자료형 일치 검사.

시퀀스·반복

시그니처설명
len(seq)길이.
range(start, stop, step)정수 시퀀스(range 객체).
enumerate(iter, start=0)(인덱스, 값) 짝.
zip(*iters)같은 인덱스끼리 묶어 튜플 시퀀스로.
sorted(iter, key=None, reverse=False)정렬된 새 리스트 반환.
reversed(seq)역순 이터레이터.
sum(iter), min(iter), max(iter)합·최솟값·최댓값.
any(iter), all(iter)하나라도 참 / 모두 참 검사.
map(f, iter), filter(f, iter)함수 적용 / 조건 통과 항목.

기타

시그니처설명
abs(x), round(x, n), pow(x, y)절댓값·반올림·거듭제곱.
help(obj), dir(obj)도움말과 속성 목록.
open(path, mode='r', encoding='utf-8')파일 열기.

C.2 문자열 메소드

메소드설명
s.upper(), s.lower(), s.title()대소문자 변환.
s.strip(), s.lstrip(), s.rstrip()양쪽·왼쪽·오른쪽 공백 제거(인자에 문자 지정 가능).
s.split(sep=None)구분자로 나눠 리스트.
sep.join(iter)문자열 시퀀스를 sep으로 잇는다.
s.replace(old, new)부분 문자열 교체.
s.find(sub), s.index(sub)위치 검색(없으면 (-1) / 예외).
s.startswith(p), s.endswith(s)시작·끝 검사.
s.isdigit(), s.isalpha(), s.isalnum()문자 종류 검사.
s.format(...), f'{var}'서식화 출력.

C.3 리스트 메소드

메소드설명
lst.append(x)끝에 한 항목 추가.
lst.extend(iter)여러 항목을 한 번에 이어 붙임.
lst.insert(i, x)위치 i에 삽입.
lst.remove(x)첫 번째로 등장하는 x 제거.
lst.pop(i=-1)인덱스 i 항목을 꺼내 반환(기본은 마지막).
lst.index(x), lst.count(x)위치·등장 횟수.
lst.sort(key=None, reverse=False)제자리 정렬.
lst.reverse()제자리 뒤집기.
lst.copy()얕은 복사(lst[:]와 동일).

C.4 딕셔너리 메소드

메소드설명
d.keys(), d.values(), d.items()키·값·쌍 뷰.
d.get(key, default=None)키가 없으면 default 반환.
d.setdefault(key, default)키가 없을 때만 추가.
d.pop(key, default)키 제거 후 값 반환.
d.update(other)다른 딕셔너리·쌍 시퀀스 병합.
d.clear()모든 항목 제거.
key in d키 존재 검사(딕셔너리에서 가장 빠른 패턴).

C.5 집합 메소드

메소드 / 연산자설명
s.add(x), s.remove(x), s.discard(x)추가·제거(없을 때 예외 / 무시).
s.union(t) 또는 s | t합집합.
s.intersection(t) 또는 s & t교집합.
s.difference(t) 또는 s - t차집합.
s.symmetric_difference(t) 또는 s \^{} t대칭차.
s.issubset(t), s.issuperset(t)부분·상위 집합 판정.

C.6 파일 입출력 패턴

# 텍스트 읽기 (한 번에)
with open('data.txt', encoding='utf-8') as f:
    text = f.read()
# 한 줄씩 처리
with open('data.txt', encoding='utf-8') as f:
    for line in f:
        print(line.rstrip())
# 쓰기 / 추가
with open('out.txt', 'w', encoding='utf-8') as f:
    f.write('hello\n')
with open('out.txt', 'a', encoding='utf-8') as f:
    f.write('one more line\n')

C.7 자주 쓰는 표준 모듈

math — 수학 함수

이름설명
math.pi, math.e, math.inf, math.nan주요 상수.
math.sqrt(x), math.pow(x, y), math.exp(x)제곱근·거듭제곱·지수함수.
math.log(x, base=e)로그(밑 지정 가능).
math.sin(x), math.cos(x), math.tan(x)삼각함수(인자 단위는 라디안).
math.floor(x), math.ceil(x), math.trunc(x)내림·올림·정수부.
math.gcd(a, b), math.factorial(n)최대공약수·팩토리얼.

random — 의사 난수

함수설명
random.random()([0.0, 1.0)) 사이 실수.
random.uniform(a, b)([a, b]) 사이 실수.
random.randint(a, b)([a, b]) 사이 정수(양 끝 포함).
random.choice(seq), random.sample(seq, k)무작위 추첨 / 비복원 추출.
random.shuffle(lst)제자리에서 섞기.
random.seed(n)결과 재현용 시드.

datetime — 날짜와 시간

함수 / 메소드설명
datetime.now()현재 시각.
datetime(year, month, day)특정 시각 생성.
dt.strftime(fmt)문자열 서식 (예: '%Y-%m-%d %H:%M').
datetime.strptime(s, fmt)문자열을 datetime으로 파싱.
timedelta(days, seconds, ...)시간 간격(덧·뺄셈 가능).

os — 운영체제 인터페이스

함수설명
os.getcwd(), os.chdir(path)현재 디렉터리 확인·변경.
os.listdir(path='.')디렉터리 안의 항목 목록.
os.path.join(a, b, ...)OS에 맞춘 경로 연결.
os.path.exists(path)파일/폴더 존재 확인.
os.makedirs(path, exist_ok=True)중간 디렉터리까지 생성.

C.8 함께 익혀 두면 좋은 빈출 패턴

패턴설명
