1. Python 공식 자료

파이썬 언어의 공식 문서, 표준 라이브러리, 그리고 언어 설계 원칙을 담은 PEP 문서들입니다.

Python Official Website
Python Software Foundation

파이썬 언어의 공식 홈페이지. 최신 버전 다운로드, 뉴스, 커뮤니티 링크를 제공한다.
python.org →
Python 3 Documentation

파이썬 3의 전체 공식 문서. 언어 레퍼런스, 라이브러리 레퍼런스, 설치 가이드, HOW-TO 문서 등을 망라한다. 파이썬 학습 시 가장 권위 있는 1차 자료이다.
docs.python.org →

파이썬 공식 튜토리얼. 언어의 기본 개념과 기능을 단계적으로 소개하며, 다른 언어 경험이 있는 프로그래머가 파이썬에 빠르게 입문할 수 있도록 구성되어 있다.
Python Tutorial →
The Python Standard Library
Python Software Foundation
2024 레퍼런스
파이썬에 기본 포함된 표준 라이브러리의 전체 레퍼런스. math, datetime, os, json 등 200여 개 모듈의 사용법을 상세히 기술한다.
Standard Library →
PEP 8 — Style Guide for Python Code
Guido van Rossum, Barry Warsaw, Alyssa Coghlan
2001 PEP
파이썬 코딩 스타일의 사실상 표준. 들여쓰기, 네이밍 컨벤션, 주석 작성법 등 가독성 높은 코드를 작성하기 위한 규약을 정의한다. 파이썬 개발자라면 반드시 숙지해야 할 문서이다.
PEP 8 →
PEP 20 — The Zen of Python
Tim Peters
2004 PEP
파이썬의 설계 철학을 19개의 경구(aphorism)로 표현한 문서. "Beautiful is better than ugly", "Explicit is better than implicit" 등 파이썬 커뮤니티의 핵심 가치를 담고 있다. import this로 확인 가능하다.
PEP 20 →
PEP 257 — Docstring Conventions
David Goodger, Guido van Rossum
2001 PEP
파이썬 독스트링(docstring) 작성 규약. 모듈, 클래스, 함수에 대한 문서화 문자열의 형식과 내용에 대한 가이드라인을 제공한다.
PEP 257 →
PEP 484 — Type Hints
Guido van Rossum, Jukka Lehtosalo, Łukasz Langa
2014 PEP
파이썬에 타입 힌트(type hint)를 도입한 PEP. 정적 타입 검사 도구(mypy 등)를 활용할 수 있게 하며, 코드의 가독성과 유지보수성을 높이는 데 기여한다.
PEP 484 →

모든 PEP(Python Enhancement Proposal)의 색인. 파이썬 언어의 발전 과정과 설계 결정의 역사를 추적할 수 있다.

2. 파이썬 입문서

파이썬 프로그래밍을 처음 배우는 독자를 위한 입문 교재들입니다.

Learning Python (5th Edition)
Mark Lutz
O'Reilly, 2013 ISBN: 978-1449355739
1,600페이지에 달하는 파이썬 종합 입문서. 자료형, 함수, 모듈, 클래스, 예외 처리 등 파이썬의 모든 기초 문법을 깊이 있게 다루며, 풍부한 예제와 연습 문제를 제공한다. 초보자부터 중급 개발자까지 레퍼런스로 활용 가능하다.
Python Crash Course (3rd Edition)
Eric Matthes
No Starch Press, 2023 ISBN: 978-1718502703
가장 인기 있는 파이썬 입문서 중 하나. 전반부에서 기초 문법을 빠르게 학습한 뒤, 후반부에서 웹 애플리케이션, 데이터 시각화, 게임 등 세 가지 프로젝트를 실습한다. 빠르고 실용적인 학습을 추구하는 독자에게 적합하다.
Automate the Boring Stuff with Python (2nd Edition)
Al Sweigart
No Starch Press, 2019 ISBN: 978-1593279929
반복적인 업무를 파이썬으로 자동화하는 방법을 다루는 실용서. 파일 관리, 웹 스크래핑, 엑셀 처리, 이메일 발송 등 실생활에서 바로 활용 가능한 자동화 기법을 소개한다. 온라인에서 무료로 읽을 수 있다.
Think Python: How to Think Like a Computer Scientist (2nd Edition)
Allen B. Downey
O'Reilly, 2015 ISBN: 978-1491939369
컴퓨터 과학적 사고방식을 파이썬을 통해 배울 수 있는 교재. 알고리즘 설계, 디버깅 전략, 분석적 사고 등 프로그래밍의 본질에 초점을 맞추며, 간결하면서도 명료한 설명이 특징이다.
Head First Python (2nd Edition)
Paul Barry
O'Reilly, 2016 ISBN: 978-1491919538
시각적이고 참여형인 Head First 시리즈의 파이썬판. 뇌과학 기반의 학습 원리를 적용하여 그림, 퍼즐, 대화 형식 등으로 파이썬 개념을 직관적으로 전달한다.
Python for Everybody: Exploring Data in Python 3
Charles R. Severance
CreateSpace, 2016 ISBN: 978-1530051120
데이터 탐색에 초점을 맞춘 파이썬 입문서. 웹 데이터 수집, 데이터베이스 활용, 데이터 시각화 등을 다루며, Coursera 강좌와 함께 세계적으로 널리 사용된다. 온라인 무료 공개 도서이다.
py4e.com →
Introduction to Computation and Programming Using Python (3rd Edition)
John V. Guttag
MIT Press, 2021 ISBN: 978-0262542364
MIT의 전설적인 컴퓨터 과학 입문 교재. 파이썬을 도구로 활용하여 계산적 사고, 알고리즘 분석, 데이터 과학의 기초를 체계적으로 학습한다. 학술적 깊이와 실용성을 겸비한 교재이다.
Python Programming: An Introduction to Computer Science (3rd Edition)
John Zelle
Franklin, Beedle & Associates, 2016 ISBN: 978-1590282755
컴퓨터 과학 입문 과정을 위한 교과서. 그래픽스를 활용한 시각적 예제가 특징이며, 프로그래밍의 기본 개념을 단계적으로 소개한다.
Intro to Python for Computer Science and Data Science
Paul Deitel, Harvey Deitel
Pearson, 2019 ISBN: 978-0135404676
컴퓨터 과학과 데이터 과학을 동시에 다루는 Deitel 시리즈의 파이썬 교재. AI, 빅데이터, 클라우드 컴퓨팅 등 현대적 주제를 풍부하게 포함한다.
Learn to Program (3rd Edition)
David Pine
Pragmatic Bookshelf, 2021 ISBN: 978-1680508178
프로그래밍을 전혀 모르는 완전한 초보자를 위한 입문서. 최소한의 전제 지식만으로 프로그래밍의 기초 개념을 차근차근 배울 수 있도록 구성되어 있다.

3. 파이썬 중급/고급서

파이썬의 고급 기법과 관용구, 성능 최적화 등을 다루는 중급 이상의 교재들입니다.

Fluent Python (2nd Edition)
Luciano Ramalho
O'Reilly, 2022 ISBN: 978-1492056355
파이썬의 고급 기법을 마스터하기 위한 필독서. 데이터 모델, 함수형 프로그래밍, 메타프로그래밍, 동시성 프로그래밍 등 파이썬다운(Pythonic) 코드를 작성하기 위한 심층적인 내용을 다룬다.
Effective Python: 90 Specific Ways to Write Better Python (2nd Edition)
Brett Slatkin
Addison-Wesley, 2019 ISBN: 978-0134853987
90개의 구체적인 조언을 통해 더 나은 파이썬 코드를 작성하는 방법을 제시한다. 각 항목은 독립적으로 읽을 수 있으며, 실무에서 흔히 마주치는 상황에 대한 모범 답안을 제공한다.
Python Cookbook (3rd Edition)
David Beazley, Brian K. Jones
O'Reilly, 2013 ISBN: 978-1449340377
파이썬 프로그래밍의 실전 레시피 모음. 자료구조, 문자열, 파일, 네트워킹, 메타프로그래밍 등 다양한 주제에 대한 검증된 해결책을 제시한다.
Python in a Nutshell (3rd Edition)
Alex Martelli, Anna Ravenscroft, Steve Holden
O'Reilly, 2017 ISBN: 978-1449392925
파이썬 언어의 핵심을 압축적으로 정리한 데스크 레퍼런스. 언어 문법, 표준 라이브러리, 확장 모듈 등을 빠르게 찾아볼 수 있도록 구성되어 있다.
Robust Python: Write Clean and Maintainable Code
Patrick Viafore
O'Reilly, 2021 ISBN: 978-1098100667
견고하고 유지보수 가능한 파이썬 코드를 작성하는 기법을 다룬다. 타입 어노테이션, 열거형, 데이터 클래스, 의존성 관리 등 코드의 안정성을 높이는 실용적인 방법을 소개한다.
Architecture Patterns with Python
Harry Percival, Bob Gregory
O'Reilly, 2020 ISBN: 978-1492052203
TDD, DDD, 이벤트 기반 마이크로서비스 등 현대적 소프트웨어 아키텍처 패턴을 파이썬으로 구현하는 방법을 보여준다.
Python 3 Object-Oriented Programming (3rd Edition)
Dusty Phillips
Packt, 2018 ISBN: 978-1789615852
파이썬의 객체 지향 프로그래밍을 깊이 있게 다루는 교재. 상속, 다형성, 디자인 패턴, 테스팅 등을 체계적으로 학습할 수 있다.
Serious Python
Julien Danjou
No Starch Press, 2018 ISBN: 978-1593278786
배포, 확장성, 테스팅 등 실무 파이썬 개발의 고급 주제를 다룬다. 오픈 소스 프로젝트에서의 실전 경험을 바탕으로 한 조언이 풍부하다.
Python Essential Reference (4th Edition)
David M. Beazley
Addison-Wesley, 2009 ISBN: 978-0672329784
파이썬 언어의 핵심 기능과 가장 많이 사용되는 라이브러리를 간결하게 정리한 레퍼런스. 숙련된 개발자가 빠르게 참고하기에 적합하다.

4. 데이터 과학 & 분석

데이터 분석, 시각화, 통계의 원리를 다루는 서적들입니다.

Python for Data Analysis (3rd Edition)
Wes McKinney
O'Reilly, 2022 ISBN: 978-1098104030
pandas 라이브러리의 창시자가 직접 쓴 데이터 분석 교재. pandas, NumPy, matplotlib를 활용한 데이터 정제, 변환, 분석, 시각화의 전 과정을 체계적으로 다룬다.
Python Data Science Handbook
Jake VanderPlas
O'Reilly, 2016 ISBN: 978-1491912058
IPython/Jupyter, NumPy, pandas, matplotlib, scikit-learn의 5대 도구를 깊이 있게 다루는 데이터 과학 핸드북. 풍부한 실습 예제와 함께 각 도구의 핵심 기능을 설명한다.
Data Science from Scratch (2nd Edition)
Joel Grus
O'Reilly, 2019 ISBN: 978-1492041139
데이터 과학의 핵심 알고리즘을 라이브러리 없이 순수 파이썬으로 직접 구현해 보는 교재. 통계, 확률, 선형대수, 머신러닝의 원리를 밑바닥부터 이해할 수 있다.
Storytelling with Data
Cole Nussbaumer Knaflic
Wiley, 2015 ISBN: 978-1119002253
데이터 시각화를 통한 효과적인 커뮤니케이션 방법을 다루는 서적. 어떤 차트를 선택하고, 어떻게 디자인해야 메시지를 명확하게 전달할 수 있는지를 실례를 통해 보여준다.
R for Data Science (2nd Edition)
Hadley Wickham, Mine Çetinkaya-Rundel, Garrett Grolemund
O'Reilly, 2023 ISBN: 978-1492097402
R 언어를 사용한 데이터 과학 입문서이지만, 데이터 과학의 워크플로우(가져오기, 정리, 변환, 시각화, 모델링, 소통)를 체계적으로 정립한 교과서적 구성으로 파이썬 사용자에게도 큰 참고가 된다.
r4ds.hadley.nz →
Naked Statistics
Charles Wheelan
W.W. Norton, 2013 ISBN: 978-0393347777
통계학의 핵심 개념을 수식 없이 일상적 사례를 통해 설명하는 교양서. 회귀분석, 중심극한정리, 가설검정 등을 직관적으로 이해할 수 있다.
The Art of Statistics: How to Learn from Data
David Spiegelhalter
Basic Books, 2019 ISBN: 978-1541618510
현대 통계학의 원리를 실제 사례를 통해 설명하는 교양서. 데이터에서 의미 있는 결론을 도출하는 통계적 사고방식을 배울 수 있다.
Practical Statistics for Data Scientists (2nd Edition)
Peter Bruce, Andrew Bruce, Peter Gedeck
O'Reilly, 2020 ISBN: 978-1492072942
데이터 과학자를 위한 실용 통계서. EDA, 통계적 실험, 회귀, 분류 등 데이터 과학 실무에서 빈번히 사용되는 통계 기법을 R과 Python 코드로 실습한다.
The Visual Display of Quantitative Information (2nd Edition)
Edward R. Tufte
Graphics Press, 2001 ISBN: 978-1930824133
데이터 시각화의 고전. 정보 디자인의 원칙, 차트 왜곡(chartjunk)의 위험, 높은 데이터-잉크 비율의 중요성 등 시각화의 기본 철학을 정립한 명저이다.
The Truthful Art
Alberto Cairo
New Riders, 2016 ISBN: 978-0321934079
데이터, 차트, 지도를 통한 정확한 커뮤니케이션을 다루는 시각화 교재. 통계적 사고와 시각적 표현을 결합하여 진실을 전달하는 방법을 탐구한다.
Fundamentals of Data Visualization
Claus O. Wilke
O'Reilly, 2019 ISBN: 978-1492031086
정보 전달력이 높은 시각화를 만들기 위한 기본 원칙을 다루는 교재. 색상 선택, 축 설계, 차트 유형 선택 등 실무적 판단 기준을 제공한다. 온라인 무료 공개이다.
clauswilke.com/dataviz →

5. 머신러닝 & AI

머신러닝, 딥러닝, 인공지능의 이론과 실습을 다루는 서적들입니다.

Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow (3rd Edition)
Aurélien Géron
O'Reilly, 2022 ISBN: 978-1098125974
머신러닝과 딥러닝의 실습 바이블. scikit-learn으로 전통적 ML을, Keras/TensorFlow로 딥러닝을 배운다. 이론과 실습의 균형이 뛰어나며, 실무 투입 가능한 수준의 프로젝트를 다룬다.
Pattern Recognition and Machine Learning
