# 으뜸 파이썬 Express — 예제 코드

이 저장소는 **『으뜸 파이썬 Express』** (생능출판사)에 수록된 파이썬 예제 코드를 제공합니다.

## 책 소개

『으뜸 파이썬 Express』는 기존 『으뜸 파이썬』의 핵심 내용을 **12개 장**으로 압축한 Express 에디션입니다. 빠르게 파이썬을 학습하고 실전에 활용하고자 하는 독자를 위해 구성되었습니다.

**[책 소개 페이지 바로가기](https://dknife.github.io/PythonExpress/)**

## 실행 환경

- Python 3.10 이상 권장
- IV부(10~12장)의 일부 예제는 추가 패키지가 필요합니다:
  ```
  pip install numpy pandas matplotlib scikit-learn flask
  ```
